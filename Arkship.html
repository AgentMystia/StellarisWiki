Arkship

Version

This article has been verified for the current PC version (4.4) of the game.
 Available only with the Nomads DLC enabled.
Tier III civilian arkship

Arkships are mobile megastructures which serve as hybrid ships and colonies for Nomadic empires. They can only be constructed by another arkship once the technology with the same name has been researched. All of them contain a colony with 40% Habitability, allow the empire to take one contract, can build ships, repair docked ships, and can have modules installed similar to a starbase as well as systems. All arkships have 100 Sublight Speed and 5 Evasion and start with 1000 Pops. They are automatically upgraded with the latest researched technologies and use mechanical components even if the empire uses a biological shipset.

Each arkship can have two leaders assigned: a governor and a captain. The governor can be any class and is assigned either from the Colony screen or the Ship Controls tab, where they are the leader on the left. The captain can only be a Commanders or Scientists and is assigned from the Ship screen or the Ship Controls tab, where they are the leader on the right. Note that leader traits are only applied to the position they are assigned: a leader with ship bonuses (i.e. Sublight Speed, Survey Speed, Fire Rate) will not apply these bonuses to the arkship when assigned as a governor.

Settled empires can construct an arkship through the Embark decision, but it will either turn the empire fully Nomadic or have the arkship be created as a subject. It is not possible to own both settled colonies and arkships as the same empire.

Selecting an arkship on the map will bring the ship actions menu and allow moving the ark. The colony can be selected from the outliner but both menus include a button to switched between them.

Available only with the First Contact DLC enabled.

Arkships cannot cloak but can have Detection Strength.

Orders

Arkship harvesting resources from a planet

Arkships can perform the following orders, which can also be automated:

  • Survey gives detailed information about a celestial object, namely deposits, habitability, planet modifiers, planetary features, and discoveries. It requires a leader assigned to the arkship. This order receives bonuses from Survey Speed modifiers.
  • Deep Scan costs 500 Operational Reserves and 90 days, includes the effects of Survey, and has a 15% chance to gain resources, 25% chance to discover new deposits, 7% chance to discover an anomaly and 3% chance to discover an archaeological site. It requires a leader assigned to the arkship and either a tier 2 arkship, an Advanced Sensor Array module, or a Subspace Imaging Array spinal mount. If a new deposit is uncovered, any Resource Depletion modifier is immediately removed. The order can be performed multiple times on the same celestial body but each successive use lowers the chances of finding anything. This order receives bonuses from Survey Speed modifiers. Habitable planets can also be deep scanned if the renowned paragon Sero the Younger is assigned to the arkship.
  • A spinal mount ability, which requires the arkship to have a spinal mount. All these orders receive bonuses from Arkship Action Speed modifiers.
    • Subspace Image System costs 10000 Operational Reserves, needs 120 for every celestial object in the system, and requires a Subspace Imaging Array. It can be done once every 5 years and only once per system. It performs a Deep Scan on the entire system where the chance to uncover a resource cache or deposits are increased to 30% each and has a 2.5% chance to create a new hyperlane to the nearest unconnected system or a wormhole if the system doesn't already have one. Hyperlanes cannot be created towards the Corrupted Avatar system or a choke point system. The chance to create a wormhole is increased to 50% if done in the Sol system.
    • Exodus Jump costs 10000 Operational Reserves and 60 days and requires an Exodus Jump Coordinator. The empire will have to place a target in the galaxy and the arkship and all owned ships in the system are sent to either the target or a random system within its scatter radius. Accuracy decreases with distance from the arkship. Ships that jumped gain +33% Fire Rate for 4 months. In both the the origin and the destination system, all fleets and stations will receive 10000 Damage and all worlds will receive 33% Devastation. If the system was owned by an empire, this will give −300 Opinion.
    • Stellar Ignition costs 5000 Operational Reserves and 500 Dark Matter and takes 180 days. It can only be done every 5 years and only once per celestial object. It can target either Gas Giants or stars as long as they don't have an anomaly. Both options will give the system −5% Maximum Habitability for all species. If used on a star, it gives it a deposit of 5 Energy if it already has such a deposit, otherwise 3 Physics. If used on a Gas Giant, it turns it into a protostar with a deposit of 10 Energy and 20 Physics but removes all moons. If used on Jupiter, Europa will survive and become a Tropical World with no blockers and the Dark Matter Monolith planetary feature.

During space battles, arkships can also be manually moved.

Available only with the Lithoids DLC enabled.

If the empire has the Terravore civic, arkships can be ordered to start the Consume World situation by right-clicking on a habitable planet and selecting the option from there.

Available only with the The Machine Age DLC enabled.

If the Empire has the Unbridled Consumption tradition, arkships can be ordered to start the Extract Nanites situation by right-clicking on a habitable planet and selecting the option from there. Doing this inside the borders of another empire gives a decaying −100 Opinion, which decreases twice faster if the system owner adopted the Nanotech tradition tree.

Resource orders

Arkships have three tiers orders that can be used to harvest resources from uninhabitable planets, asteroids, and gas giants, as long as they do not have an anomaly. Using there orders on celestial objects with deposits within the borders of another empire that have deposits on them gives an Opinion penalty unless the system has a harvest contract. Resource orders receive bonuses from Arkship Action Speed modifiers.

After any of these orders is used, the celestial object will gain the Resource Depletion modifier, which prevent any of these orders from being used on it again and also reduce the resources produced by mining stations by −75%. Resource deposits from these planets will appear yellow on the map.

The orders can be performed from the ship controls, in which case if the system map is open every celestial object will display the resources that will be harvested. The order can also be performed from the right-click menu on the galaxy map, in which case is will be queued to every celestial object in the system in the order of resource value and the tooltip will display the total resources that will be harvested.

Tiers and types

There are three tiers and three types of arkships.

  • Civilian Arkships give the empire an additional +2500 Operational Reserves Capacity, gain +5% Resources from Harvest Actions per tier and use the same default names as colony ships
  • Scientific Arkships gain +1 Sensor Range and Hyperlane Detection Range and +5% Survey Speed per tier and use the same default names as science ships
  • Military Arkships gain +5% Fire Rate and Weapons Range per tier and use the same default names as juggernauts
