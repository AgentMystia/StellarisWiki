Makes text bold and colored, using game text colors or a custom color. Note that the preset shades of red and green in this template are different from those in {{Red}}, {{Green}}, {{Blue}}, and {{Yellow}}.

Usage

The first parameter sets the color according to the list below; the second parameter sets the text; the third parameter applies a background effect if defined

{{color|red|text you want red}} gives text you want red

{{color|yellow|text you want yellow|1}} gives text you want yellow

List of colors

Stellaris defines text colors using standard 255 RGB; the following colors are pre-defined in this template and can be found in Stellaris/interface/fonts.gfx. Note that keys are case sensitive.

Custom colors can be used by defining any valid CSS color value. For example, {{color|#cc11cc|hexcode}} gives hexcode
