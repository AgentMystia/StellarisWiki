Summary of Possible Outcomes

These are all of the possible end results for terminating the rift. This table does not include rewards given for intervening chapters.

Chapter Outcome
7-A
  • astral thread reward (~50, ~100, or ~150) based on the "time passed" variable which increments with some choices.
  • Time Crystal relic.
  • If leader species has the Latent Psionic trait, leader gains the Psychic trait.

  • large astral threads reward (140, 150, or 160)

Unique Mechanics

For this rift several chapters have a "right" choice that proceeds to the next chapter and a "wrong" choice that repeats the current one. The game silently tracks a variable for the "time passed" which is incremented each time a choice is made. If the time passed reaches a value of 3 then the Time Crystal outcome becomes impossible. If the rift ends with the Time Crystal outcome then the astral threads reward amount is a function of the time passed.

The fastest sequence of choices for the Time Crystal outcome is as follows:

  • Blue button
  • Pentagon button
  • Six dots button

No more than one error can be made and still obtain the Time Crystal outcome, with the following exception: If the leader species is Toxoid or has the Aquatic or Waterproof trait then the Time Crystal outcome can still be reached after making 2 errors. Because the final astral threads reward is based on the time passed variable and not the number of chapters visited, the player can maximize the amount of astral threads received from the rift by intentionally selecting the "wrong choice" in The Second Chamber if Toxoid or in The Third Chamber if Aquatic or Waterproof.

Options which indirectly force or preclude any final outcomes of the rift are noted in blue.

Chapters which terminate the rift are noted in red.

Because the order in which events are encountered depends on the player's choices, chapter numbers in this table may differ from the chapter numbers assigned to the events in-game. This table assigns chapter numbers based on the optimal progression of events (in which no errors are made), while the in-game rift dialog window will assign chapter numbers based on the order in which events are encountered.
