Template:Store

Use this template to place links to the Paradox Interactive or Steam store.

Usage

To place a Paradox Interactive store link:

  • Open the desired store page, for example: https://www.paradoxinteractive.com/games/stellaris/add-ons/stellaris-utopia
  • Copy everything after stellaris/. In the example above it would be: add-ons/stellaris-utopia
  • Paste the copied part into the template. The syntax should look as this: {{Store|pdx|add-ons/stellaris-utopia}}
  • Result: Paradox

To place a Steam store link:

  • Open the desired store page, for example: https://store.steampowered.com/app/553280/Stellaris_Utopia/
  • Copy the offer type (app, bundle, and etc.) with the associated id. In the example above it would be: app/553280
  • Paste the copied part into the template. The syntax should look as this: {{Store|steam|app/553280}}
  • Result: Steam
