Usage

{{path|whatever/path}} generates Stellaris/whatever/path

OPTIONAL {{{full|}}}

Prepends full path to Stellaris {{path|full=1|whatever/path}} generates C://Steam/SteamApps/common/Stellaris/whatever/path

OPTIONAL {{{custom|}}}

Prepends nothing. {{path|custom=1|whatever/path}} generates whatever/path
