This template can be used to generate icons for ship components and weapons. Use {{Mutation}} for Space Fauna components.

Component images can be found here: Category:Ship part icons

Parameters

Parameter Description Required Default
1 Ship part icon name, minus "ship part" required none
2 Localized name, if different optional none
type Component type, one of:
  • weapon
  • utility
  • core

Also functions as page link selector

optional core
slot Component size, for weapon or utility components, any of:
  • a
  • g
  • h
  • l ('ell')
  • m
  • p
  • s
  • t
  • w
  • x
 optional a for utility type

s for weapon type

w Sets icon size optional 58px
iconify if set to "yes", appends text of parameter 2 or 1 optional none

Examples

Wikitext Procudes
{{Component|Subspace Drive}}
{{Component|Subspace Drive|iconify=yes|w=24px}} Subspace Drive
{{Component|laser 1|Red Laser|type=weapon}}
{{Component|afterburners 2|Advanced Afterburners|type=utility|iconify=yes|w=24px}} Advanced Afterburners
{{Component|Perdition Beam|slot=t|type=weapon|iconify=yes}} Perdition Beam
