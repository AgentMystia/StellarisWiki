Template:Planet modifier

[view] [edit] Template documentation
This documentation is transcluded from Template:Planet modifier/doc. (edit | history)

This template displays a planet modifier icon with a green, yellow or red frame.

Parameters

Parameter Description Required Default
1 Modifier picture, with or without "pm_" required none
2 / frame
  • Modifier frame:
    • 1 / green = green
    • 2 / yellow = yellow
    • 3 / red = red
 required yellow
3
  • Modifier name
  • Will automatically generate a link
 optional none
link
  • Links to a specified page
  • Will also set hovertext if parameter 3 is not defined
 optional
  • Planet modifiers#[3]
  • none if parameter 3 is not defined
iconify
  • If set to yes, it will append the icon with the text in parameter 3
  • If used, will make icon default to 24px instead.
 optional no
w Size of the planet modifier icon optional
  • 60px
  • 24px if iconify is used

Examples

Wiki text Produces
{{Planet modifier|pm_archaeology|3}}
{{Planet modifier|crime|red|Criminal Underworld}}
{{Planet modifier|machine_sapience|yellow|Zen World|iconify=yes}} Zen World
