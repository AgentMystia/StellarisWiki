Template:Tradition/doc

This is the documentation page for Template:Tradition.
When it is viewed directly, links using variables may appear broken; do not replace these with hardcoded page names or URLs.


This template can be used to generate icons for traditions.

Tradition images can be found here: Category:Tradition_icons

Parameters

Parameter Description Required Default
1 Tradition tree required none
2 Tradition name
  • Determines the icon image in Tradition [1] [2].png format
  • If set to icon or iconify, will apply formatting settings from parameter 3
  • If set to finisher, the phrase "finisher" will be appended to the hover text and iconify text
 optional icon
3
Icon format
  • If set to icon, sets default size to 24px
  • If set to iconify, sets default size to 24px, append the icon with the tradition name
 optional no resize, no text
w Sets icon size
  • If either 3 or 2 is set to icon or iconify, defaults to x24px
 optional no resize
image Icon image, overrides parameter 2 optional Tradition [1] [2].png
name Tradition name, overrides parameter 2 in hover text and iconify text, but not in image file. optional [2]

Examples

Wikitext Procudes
{{tradition|expansion}}
{{tradition|Expansion|iconify}} Expansion
{{tradition|expansion|Galactic Ambition}}
{{tradition|expansion|Galactic Ambition|icon}}
{{tradition|expansion|Galactic Ambition|iconify}} Galactic Ambition
{{tradition|modularity|5|name=Advanced Tools}}
{{tradition|modularity|5|iconify|name=Advanced Tools}} Advanced Tools
