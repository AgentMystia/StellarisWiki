Template:Version column/Patch

 This is a subtemplate of Template:Version column.

[[Patch {{{1}}}|{{{1}}}]]

This template generates links to patch note pages. Shortcut call: {{Patch}}

Parameters

Parameter Use
1 Main patch number, e.g. 1.1, 1.5, etc.
2 Minor patch number, e.g. 1, 5, 13, etc.

Examples

  • {{Patch|3.10}} gives 3.10
  • {{Patch|2.3|1}} gives 2.3.1
