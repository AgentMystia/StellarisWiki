Template:Tradition namebox/doc

This is the documentation page for Template:Tradition namebox.
When it is viewed directly, links using variables may appear broken; do not replace these with hardcoded page names or URLs.

Template used to generate tradition nameboxes to use on the Traditions page.

Parameters

Parameter Description Required Default
1 Tradition tree, e.g. "Discovery", used to generate icon required —
2 Tradition name, e.g. "To Boldly Go", used to generate icon required —
name Displayed tradition name, unnecessary if it matches 2 optional [2]
hive Tradition name for hive empires, if applicable. Displays alternate names in a tooltip. optional —
machine Tradition name for machine empires, if applicable. Displays alternate names in a tooltip. optional —
inline If set to "yes", displays the tradition as an inline block (important for centering it in a cell) optional —
effects Lists tradition effects below the namebox. optional —

Examples

{{tradition namebox|Discovery|To Boldly Go}} will generate:

To Boldly Go

{{tradition namebox|Discovery|To Boldly Go|hive=Sensory Tendrils|machine=Exploration Drones}} will generate:

To Boldly Go
Sensory Tendrils
Exploration Drones
