UserInfo can be used by anyone wanting to provide some information about themselves in a straightforward manner

Usage

This infobox may be added by copy-pasting the template as shown below into an article. 

{{UserInfo 
| forum-nick = 
| forum-id = 
| location =   
| join =  
| roles =  
| avatar = 
| image-size =
}}

Parameters

Parameter Description Default
forum-nick Username on the Paradox forums,
if different from on the wiki		 Wiki username
forum-id Forum user page id number:
e.g.1234 or
paradox-interactive.1
location User's location Unspecified
join User's join date Unspecified
roles User's wiki roles Contributor
avatar An image file to represent the user
image-size Size of the avatar file 60px
