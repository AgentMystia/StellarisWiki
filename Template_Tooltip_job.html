Template:Tooltip job

This documentation is transcluded from Template:Tooltip job/doc. (edit | history)


Premade tooltips for jobs variants.

Parameters

Parameter Description Required Default
1 Sets a premade tooltip from the Samples section required —
2 Text inserted between job name and icon. optional —
text Overrides the underlined text, including the icon. optional {{icon|[1]|24px}} [2] [1]
plural
  • If set to no, does not pluralize the job names
  • If set to yes, pluralizes the jobs names in the tooltip and the underlined text
  • If set to head, only pluralizes the jobs name in the underlined text
  • If set to body, only pluralizes the jobs names in the tooltip
 optional no
(yes if parameter 2 is used)
exclude Specifies a job group to exclude (see more info in Samples section) optional —
include Specifies a job group to include (see more info in Samples section) optional —
index Allows tooltip-in-a-tooltip functionality, only set to 2 or 3 optional —
