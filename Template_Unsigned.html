This template should always be substituted, i.e. use {{subst:Unsigned}}.

This template can be used to label unsigned comments on talk pages.

Parameter

Parameter Description Default Status
1 User name or IP address of the user empty required
2 Timestamp from the edit history empty optional

Please add the correct time zone to the time stamp.

Example

{{subst:Unsigned|Meneth|16:52, 16 March 2015 (CET)}}

yields
