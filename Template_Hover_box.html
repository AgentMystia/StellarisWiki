Template:Hover box

[view] [edit] Template documentation
This documentation is transcluded from Template:Hover box/doc. (edit | history)

This template creates dotted underlined text with a hover box.

Parameters

Parameter Description Default Status
1 the text in the hover box – required
2 dotted underlined text – required
link link=yes enables a link to the dotted underlined text no optional

Example

{{Hover box|This province attribute was introduced with patch 1.12.|Development}}
Development 
{{Hover box|This province attribute was introduced with patch 1.12.|Development|link=yes}}
Development
