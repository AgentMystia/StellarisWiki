Game modification: Inclusive Civics

Type

Author(s)

Status

Steam workshop
Link

Inclusive Civics is a mod that relaxes the restrictions for most civics, allowing for more diverse combinations in the empire creator. The AI will continue to use the requirements from vanilla.

Changed civics

If a civic is not listed here, it is unchanged from vanilla.

Unlocked civics

The following civics can now be added or removed after game start:

  • Inward Perfection
  • Fanatic Purifiers
  • Barbaric Despoilers
  • Idyllic Bloom / Mycorrhizal Ideal
  • Anglers / Trawling Operations
  • Criminal Heritage
  • Devouring Swarm / Terravore
  • Rogue Servitor
  • Determined Exterminator
  • Driven Assimilator
