Species/Toxoid

Note: This is a subpage to list Species portraits.

Available only with the Toxoids DLC enabled.

Toxoids have drawn inspirations from three broad areas: High Tech Survivalism, Wasteland Toxpunk, and Protective Fashion. Their character designs reflect these influences. Some portraits of this class feature different models for each gender.
