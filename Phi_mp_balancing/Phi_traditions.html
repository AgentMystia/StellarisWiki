Prosperity

The Prosperity tradition tree focuses on various empire-wide bonuses and is available to all empires.

  • +20% Mining Station Output
  • −10% Building and District Upkeep
  • −10% Building and District Build Cost
  • +25% Planet Build Speed
  • +5% Specialist and Complex Drone Output
  • +1 Clerk per City District if not Gestalt Consciousness
  • +1 Clerk per Trade District if not Gestalt Consciousness
  • +3 Clerk per Residential Arcology if not Gestalt Consciousness
  • +5 Clerk per City Segment if not Gestalt Consciousness
  • +5% Monthly Energy Credits if Gestalt Consciousness
  • +1 Housing per City/Hive/Nexus District if not Void Dwellers
  • +3 Housing per Residential Arcology if not Void Dwellers
  • +5 Housing per City/Hive/Nexus Segment if not Void Dwellers
  • +1 Building Slot for Orbital Habitats if Void Dwellers
+5% Resources from Jobs
+5 Stability on all planets
