Template to generate civic icons.

Parameters

Parameter Description Required Default
1 Civic icon, uses File:Civic [1].png format required —
2 Civic name, used to set links and hover text optional [1]
w Icon size, max size for civics is either 58px or 29px. optional 24px
link Icon link.
If not used, links to the Civics page, with parameter 2 as anchor link		 optional Civics#[2]
council If set to yes, the icon links to the Council page instead, with parameter 2 as the council position name. optional —
iconify If set to yes, appends the text of parameter 2 to the icon. optional —
page Input key determines which page is linked to; keys:
  • standard > Civics
  • mi/machine/machine intelligence > Machine intelligence civics
  • hm/hive/hive mind > Hive mind civics
  • corp/corporate > Corporate civics
  • council > Council
 optional standard

Examples

{{Civic|Parliamentary System}} will generate:

{{Civic|Parliamentary System|iconify=yes}} will generate:
Parliamentary System

{{Civic|parliamentary system|Speaker of Parliament|page=council|iconify=yes}} will generate:
Speaker of Parliament

{{Civic|Franchising|page=corp|iconify=yes}} will generate:
Franchising
