Template to generate job icons.

Parameters

Parameter Description Required Default
1 Job icon, uses File:Job [1].png format required —
2 Job name, used to set links and hover text optional [1]
w Icon size. Max size of job icons is 30px. optional 24px
iconify If set to yes, appends the text of parameter 2 to the icon. optional —
link Icon link.
If not used, links to the Jobs page with parameter 2 / 3 as anchor link		 optional Jobs#[2]

Examples

{{Job|Technician}} will generate:

{{Job|Technician|iconify=yes}} will generate: Technician

{{Job|Technician|Tech-Drone|iconify=yes}} will generate: Tech-Drone
