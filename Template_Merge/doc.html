This is the documentation page for Template:Merge.
When it is viewed directly, links using variables may appear broken; do not replace these with hardcoded page names or URLs.

Merge can be used to highlight articles that are candidates for merging with other articles.

Contents

  • 2 Parameters

  • 4 Example

Usage

This table may be added by copy-pasting the template as shown below into an article. 

{{Merge
| Merger = }}

Parameters

Parameter Description Default Status
Merger The article it can merge into. – required

Effects

Puts a hint onto the page concerning article merging and adds it to the Category:To be merged.

Example

{{Merge
| Merger = Stellaris Wiki}}

Will display this:

This article is a candidate to be merged with Stellaris Wiki.
Please help to improve the wiki by taking part in the discussion of the proposed merger.
