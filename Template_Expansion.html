Expansion should be used to indicate which expansion of Europa Universalis 4 must be enabled to use the described content.

Usage

This infobox may be added by copy-pasting the template as shown below at the beginning of a section. 

{{expansion|X}}

Substitute the value for X.

Parameters

Parameter Explanation Default Status
1 To specify which expansion must be enabled empty required
2 To specify a second expansion which can be enabled instead of the first empty optional
3 To specify a third expansion which can be enabled instead of the first two empty optional
4 To specify a fourth expansion which can be enabled instead of the first three empty optional
5 To specify a fifth expansion which can be enabled instead of the first four empty optional
small Same as {{ambox}}. left optional
req Can be used to replace the "or" with "and". or optional

Values

The above parameters can have the following values:

Parameter Values Explanation
1, 2, 3, 4, 5 DLC value DLC that must be enabled
small left For sections (default)
no For articles
req or Requires either DLC (default)
and Requires all listed DLC

Examples

Page notification

<-- top of the page -->
{{expansion|utp|small=no}}
<-- content -->
 Available only with the Utopia DLC enabled.

Section notification

<-- top of the section -->
{{expansion|utp}}
<-- content -->
Available only with the Utopia DLC enabled.

Multiple DLCs

{{expansion|utp|mgc}}
Available only with the Utopia or MegaCorp DLC enabled. 
{{expansion|utp|mgc|req=and}}
Available only with the Utopia and MegaCorp DLC enabled. 
{{expansion|utp|snd|tma|bio}}
Available only with the Utopia, Synthetic Dawn, The Machine Age, or BioGenesis DLC enabled.
