This template can be used to generate icons for planetary deposit blockers.

Blocker images can be found here: Category:Planet feature icons

Parameters

Parameter Description Required Default
1 Blocker icon name, with or without "d" required none
2 Localized name, if different optional none
link Alternate link optional Planetary features#2/1
w Sets icon size optional 98px
iconify if set to "yes", appends text of parameter 2 or 1 optional none

Examples

Wikitext Produces
{{Blocker|d crater}}
{{Blocker|crater|Bomb Crater}}
{{Blocker|crater|Bomb Crater|iconify=yes|w=30px}} Bomb Crater
