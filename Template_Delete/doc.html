Template:Delete/doc

This is the documentation page for Template:Delete.
When it is viewed directly, links using variables may appear broken; do not replace these with hardcoded page names or URLs.

Delete can be used to highlight candidates for deletion. This template calls for discussion. Tagged articles are added to the category: Candidates for deletion.

Contents

  • 2 Usage
  • 3 Updating this template
  • 4 Examples

Parameters

Parameter Description Default Status
1 Reason for the deletion request and signature (~~~~) empty required

Usage

Please insert the template at the top of the page! 

{{Delete|<--insert here the reason for the deletion request--> -- ~~~~}}

Updating this template

You shouldn’t have to.

Examples

{{Delete|obsolete with patch 10.0 - ~~~~}}

will display:

This article is a candidate for deletion.
  • Discuss if it should or should not be deleted on this page’s talk page.
  • Reason: obsolete with patch 10.0 – Lillebror (talk) 09:30, 29 December 2014 (CET)
 



{{Delete}}

will display:

This article is a candidate for deletion.
  • Discuss if it should or should not be deleted on this page’s talk page.
  • Remember: explain why you marked this page for deletion on the talk page
  • To anyone: if there is no explanation why this is a candidate for deletion on the talk page or on the template itself, feel free to delete the template.
