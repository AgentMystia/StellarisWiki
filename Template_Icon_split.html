Template:Icon split

[view] [edit] Template documentation
This documentation is transcluded from Template:Icon split/doc. (edit | history)


This template can be used to create split icons from those defined in {{icon}}. It is intended for use with icons that use the different backgrounds with the same base icon, such as fanatic and non-fanatic ethics or authority swapped civics.

Parameters

Parameter Description Required Default
1 First part of the icon required —
2 Second part of the icon optional —
3 Third part of the icon optional —
4 Fourth part of the icon optional —
w Width of the icon in px, with or without "px"
Cannot use x#px format.		 optional 24px
iconify Appends the text of the specified parameter to the icon
Defaults to 1 if set to "yes"		 optional —
cross
  • Changes split style to cross split instead of vertical split. Intended for use with 4 icons.
  • When used, the entire paragraph must be put in <div>, <span>, or a table cell to prevent unwanted line breaks
  • Might not work correctly if the icon's width and height don't match.
 optional —
image[1-4] E.g. image1=[[File:Default.png|24px]]
Replaces the specified part of the icon with a selected file. 		optional —

Examples

Wiki text Produces
{{icon split|Spiritualist|Fanatic Spiritualist|iconify=yes}} Spiritualist
{{icon split|Anglers|Trawling Operations|iconify=yes}} Anglers
{{icon split|Democratic|Oligarchic|Dictatorial|Imperial|w=28px}} Non-Corporate Non-Corporate
{{icon split|Catalytic Processing|catalytic processing corporate|catalytic processing hive dark|catalytic processing machine|cross=yes|w=34px}} Catalytic Catalytic
{{icon split|City District|Hive District|Nexus District}} housing district housing district
{{icon split|image1=[[File:Spiritualist.png|24px|link=Spiritualist]]|image2=[[File:Fanatic Spiritualist.png|24px|link=Fanatic Spiritualist]]|w=24px}}
