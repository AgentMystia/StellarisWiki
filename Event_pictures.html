Event pictures

Version

This article has been verified for the current PC version (4.4) of the game.

The sprite information for the event picture graphics are defined in Stellaris/interface/eventpictures.gfx with the following format: 

 spriteTypes = {
 	spriteType = {
 		name = "GFX_evt_alien_nature"
 		texturefile = "gfx/event_pictures/alien_nature.dds"
 		masking_texture = "gfx/interface/situation_log/event_mask.dds"
 		alwaystransparent = yes
 	}
 }

If you'd like to mod in your own event pictures, create a new file mymod_eventpictures.gfx and add your picture like: 

 spriteTypes = {
 	spriteType = {
 		name = "GFX_evt_my_picture_name"
 		texturefile = "gfx/event_pictures/my_picture_name.dds"
 		masking_texture = "gfx/interface/situation_log/event_mask.dds"
 		alwaystransparent = yes
 	}
 }

You don't have to put your pictures in the Stellaris/gfx/event_pictures folder as long as you supply the correct path, but doing so helps keep things organized.

Picture format

The graphics for the event pictures are DDS files, 450x150 pixels (Aspect ratio 3:1), 32bpp RGBA. For origin thumbnails, the dimensions are 220x115 pixels.
