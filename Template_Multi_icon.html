Template:Multi icon

Displays multiple consecutive icons, defined in Template:Icon. Uses w parameter to set icon size, otherwise uses default icon size.

For example, {{multi icon|fanatic purifiers|devouring swarm|determined exterminator|w=24px}} Genocidal will display:

Genocidal

Supports up to 10 icons.
