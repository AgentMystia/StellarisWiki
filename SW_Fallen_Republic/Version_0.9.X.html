0.9.0

Version 0.9.0 was the initial release of Star Wars: Fallen Republic and released on 2020-08-19

0.9.1

Version 0.9.1 was released on 2020-08-28

New

  • added a new Weapon Glossary to the Ship Designer
  • added new deposit modifiers for Anaxes War College and Galactic Senate Building
  • added naval cap size to defense platforms
  • added various manufacturers (deposits on special shipyard planets like Kuat, Mon Cala, Rothana, Raxus, Bothawui, etc. which give you a small empire-wide bonus to your fleet)
  • added Imperial Advisor voice
  • added droid foundry on Geonosis
  • added feedback events for special technologies (Cloning, Clone Armies, Gene Tailoring, Death Star techs, Battledroid techs)
  • added food upkeep to pop growth jobs
  • added Encourage Planetary Growth monthly cost decision
  • added a Kaminoan formation event
  • added pirate Liberator cruiser
  • Legacy System for all corvette sized ships
  • added an event that makes Sith leaders force-sensitive
  • new building icons
  • added info event for the sender of the asteroid from Hammer Station
  • added a policy to disable force ascension if you don’t want to become Jedi or Sith
  • integrated the mod SWFR Room Replacement
  • added autojoin GCW via origin (if possible)
  • added drag-in GCW event for the rebels and the GE
  • new Legacy system for frigates
  • added Ord Trasian, Rothanian, Geonosian Hive, and Fedallean as minor factions
  • added CIS advisor
  • added missing Windup/down description to Trade Routes tooltips
  • added Go To location button on the Iego starting event.

Changes

  • improved country triggers, era perks are more variable on AI countries now (military countries want military perks, diplomatic countries want trade perks more, etc.)
  • improved Rebel Government Authority and Monarchy Authority
  • buffed the Starforge
  • increased the duration of the Jedi/Sith Ascension
  • various sound adjustments
  • changed playable Fallen Empires to start with only one fleet
  • updated Mon Cala name list
  • renamed Armageddon bombardment to Base Delta Zero
  • updated the credits systems
  • excluded more important planets from the habitability list
  • excluded cloud cities from the habitability list
  • Trandosha now spawns with some Wookie slaves
  • Separatist Faction now starts with the Battledroids tech
  • increased fleet power required for the AI to consider attacking a system
  • buffed Nihilus bombardment
  • reworked Multiplayer Settings (removed unused ones)
  • changed the Kingdom of Zeltron:
    • from Imperial Authority to Monarchy
    • from an Irenic Monarchy government to a Representative Monarchy one
    • swapped the Authoritarian ethic with the Egalitarian
    • swapped the Limited Telepathy trait with Zeltron Empathy (Happiness +10% / Amenities From Jobs +20%)
  • removed the monthly pulse on institution buildings (waiting for Paradox to fix it)
  • changed Increased Luxury decision cost to monthly
  • changed galaxy look (smaller stars, less bright hyperlanes)
  • changed galaxy and fleet zoom steps
  • changed the AI to care more about its admin cap
  • rebalanced combat computers
  • rebalanced Hapan architecture components
  • changed the habitability system to remove the deposits from the planets that are turned uninhabitable
  • changed habitability system to make the planet uncolonizable
  • changed alliance laws related to kicking members
  • changed alliance laws related to inviting members
  • removed Imperial only Legacy System
  • removed the limit per planet for unity/amenities producing buildings
  • changed all housing buildings to a limit of 1 per planet
  • added a small amount of naval capacity from the number of pops to remain consistent with the tooltip
  • replaced Imperial Escort Carrier with Tonnfalk Carrier
  • changed the Mandalorian Clans from Imperial authority to Dictatorial
  • increased by 10 the required intercepted trade value for pirates to spawn
  • changed vanilla pirate fleets composition
  • disabled popups for the player when AI countries were signing trade/science agreements, NA/commercial pacts and rivalries
  • changed shipyard queued max capacity
  • removed old Legacy system
  • disabled migration treaties

Fixes

  • fixed choosable Hutt Civic and Rebel Government
  • fixed Executor weapons autocomplete
  • fixed some army icons
  • fixed empire starting fleets being over naval capacity
  • fixed foreign aid planet modifier icon
  • fixed missing CIS dreadnought weapons
  • fixed duplicate space monster name (Purrgils etc)
  • fixed wrong modifier name for the Isotope-5 Reactor
  • corrected Mandalorian “Gathering Recruits” description
  • fixed Kyber Mine building lacking a job
  • disabled Dead God vanilla event
  • fixed a bug where habitats got planetary modifiers
  • fixed Starforge bathing in the star
  • removed exotic gas as a reward for killing Purrgils
  • fixed missed beam length
  • removed the possibility of researching Death Star technologies
  • fixed the Galactic Senate button
  • fixed the percent of amenities you get from Charismatic, Domestic Protocols, and Repugnant traits (it’s 20% for all eligible jobs now)
  • fixed an edge case of captured shipyards not flipping back when recaptured
  • fixed frigate modifiers being applied to corvettes
  • improved performance for tech decision (shields, armor, etc.)
  • fixed an incorrect value for the hull modifier in the ship specialization events
  • reworked War In Heaven (without the option to drag in other empires besides alliance members)
  • fixed a text overflow in the advisor UI related to War In Heaven
  • fixed the tooltip for the adoption of the Imperial Military tradition
  • fixed Fuel Production Base not being upgradeable to HQ
  • fixed the DS finishing event location button
  • fixed Desevro planet name and type when player-owned
  • fixed Nebula refineries
  • fixed Ninzam planet name
  • fixed having to rebuild Clinics/Hospitals when changing between Bacta and Spice
  • replaced vanilla asteroid event text with something more fitting to Star Wars
  • fixed Ordnance Edict upkeep
  • fixed Institutions flag/cost localizations
  • fixed vanilla resource trigger
  • fixed Hutt Elite civic description
  • fixed portraits swapping after modifying the species
  • fixed the Nu Enclave not auto-deleting when it's lost the last planet
  • fixed deficit event triggers
  • fixed city districts showing on minor factions capital after invading when the planet was an ecumenopolis

0.9.3

Version 0.9.1a was released on 2020-11-27 with checksum eda48916. It reworked how the Rebellion was done. The way the Rebel Alliance is formed will be different based on what galaxy map is used.

For NON-CANON maps you will get the option to form the Rebel Alliance on the first yearly pulse after you’ve finished the Rebel Alliance Tradition tree.

For CANON maps you will get the option to form the Rebel Alliance on the first yearly pulse after you gathered enough support and you’ve finished the Rebel Alliance Tradition tree.

New

  • 2.8.1 compatibility update
  • DX11 compatibility update
  • added a habitability slider in the galaxy settings (values over 1x are normalized to 1x)
  • added a switch in the galaxy settings for the First Galactic Empire
  • added custom war rules for countries with compound adjectives
  • added Mon Calamari advisor and greetings
  • added InterGalactic Banking Clan advisor

Changes

  • updated sector focuses to take into account Industrial, Heavy Industrial, Mining, Gas Mining, and Financial Districts
  • changed the Relic view from a 3x2 grid to a 4x2 one
  • changed the Habitability System to work with the habitability slider
  • changed the First Galactic Empire event to use the switch from galaxy settings
  • improved economic AI

Fixes

  • fixed Financial Districts not counting towards your energy mandate goal
  • fixed Industrial Districts and Heavy Industrial Districts not counting towards your mining mandate goal
  • fixed countries missing adjectives
  • fixed the Kuat Drive Yards Raid event having flipped option texts
  • fixed wars missing participants names
  • fixed shield regen utility issue
  • fixed reactor booster requirements
  • fixed building upgrade cost being cut off in planetary view

Rebellion Rework

  • Rebel Partisan Bases will produce rebel points (support) on a yearly basis
  • rebellion events will start firing in 7962
  • the number of rebel points you get from events was increased
  • you get more rebel points from events if you don’t own Megacorp and can’t build outposts
  • a tooltip was added to event options to let the player know which actions grant rebel points
  • the rebel points requirement to form the Rebel Alliance is different based on canon map size (higher for bigger maps)
  • increased the rebel points requirement to form the Rebel Alliance


0.9.35

Version 0.9.35 was released on 2021-07-04 with checksum 9c297bf. It updated the mod to be compatible with Stellaris Patch 3.0

3.0 Update & 3.0.3 Patch

  • 3.0 Compatibility Update
  • Disabled new vanilla districts.
  • Disabled new vanilla buildings.
  • Disabled select crisis game setting.
  • Disabled pop growth sliders in Game Settings.

Nemesis DLC

  • Disabled operations for now, will be readded in later patch.
  • Disabled “become the crisis”.
  • Disabled Galactic Custodian / Emperor.
  • Disabled music.
  • Disabled imperial ship- set.

New

  • Reworked institution buildings.
    • As on_destroy trigger does not work, we have reworked all institution buildings, stopping the exploit of building multiple different institution buildings on one planet.
    • Tier 1 buildings can now be built once on all planets, but have planetary build restrictions.
    • Tier 2 can only be built on capitals, and only once.
    • Institution buildings will now be destroyed or switched, if possible, upon switching institution or conquered in war.
    • Some of the default institution buildings now provide ethic attraction depending if you choose a military, research institution etc.
    • Criminal/megacorp empires no longer have institution buildings on their outposts.
  • Added Commander Gaunt event.
  • Added new main menu buttons.
  • Added Rocket Marauder ship section.
  • Added carrier ship behavior to some ships.
  • Added auto research as starting technology.
  • Added Ewoks to the Forest Moon of Endor.
  • Added Embassy requirements for most agreements to stop spam.
  • Added new turbolaser bolts.
  • Added new blaster bolts (Fighter/PD).
  • All projectiles (Rockets, Turbo, Ion, Vong, etc) explode on miss.

Performance Changes

  • Halved pop count on worlds with Minor Factions
  • AI will no longer build more than 5 habitats.
  • AI will no longer build more than 5 deep space stations.

Changes

  • Changed Hapan ethic for lore accuracy.
    • Pacifist > Militarist
  • Changed Galacian pop traits and system planet types in preparation for Galacian takeover.
  • Updated portraits for Ferroans, Galacians, Mawan, Oreenians, and the Rakata.
  • Missile models replaced with new ones.
  • Embassy action changed to require good relations.
  • ISD I and Mando moving exhaust particle made more awesome.
  • Disabled Rogue Providence cantina event. Known for causing lag/desyncs upon spawn.
  • Removed Agriculture District from oasis and sandsea worlds.
  • Enabled shortcut to cycle through planets “tab”.
  • Embassies require good relations to establish.
  • Sector Creation tooltip updated.

Balance

  • Balanced ship upgrade cost reduction modifiers. Was causing an upgrade exploit.
    • Reduced and removed modifiers from most techs.
    • Removed modifiers from all shipyards.
  • Balanced "its a trap" cantina event.
    • Influence penalty reduced from - 150 to - 50.
  • Heavily reduced special district job output multiplier modifiers.
  • Heavily reduced building job output multiplier modifiers.
  • Increased Death Star speed by 20%.
  • Balanced changes for claims cost reduction.
  • Gravity Well no longer available on asteroids.
  • Capital sector range decreased to 5.

Fixes

  • Fixed Mon Mothma instantly dying of old age on losing election.
  • Fixed Hammer station.
    • Hammer station will now no longer be distracted by incoming fire.
    • Increased health 12000 - > 25000.
    • Increased speed by 100%! 1 - > 2.
    • Increased spawning distance from planet 40 - > 60.
    • Hammer station is now able to target capital planets.
    • (Capital planets will not get destroyed if the asteroid hits, only severely damaged).
  • Fixed ship retirement system and improve UI experience.
    • The Chiss Vigilance can now be retired correctly.
    • Window will now no longer close upon retiring or reinstating a ship design.
  • Fixed CIS and CSA food modifiers not being given in correct amount.
  • Fixed Bothawui shipyard being added to the Hapan system. instead of the Hapan Shipyard.
  • Fixed Gargon and Sartinaynian system spawning with planet names "Random".
  • Fixed not being able to leave the galactic empire.
  • Fixed Kissai force sensitive bug.
  • Fixed create vassal UI error.
  • Fixed create federation overlapping text.
  • Fixed/removed ability to enact fleet contribution laws from all federations. Did nothing as federation fleets are disabled.
  • Fixed Vong transport ships GFX box error.
  • Fixed ship- names that were too long.
  • Fixed pre- triggers for laser cells, plasma cells, starfighter parts, ordnance, and hyperfuel jobs.
  • Fixed bug with Chiss and Banking Clan VO.
  • Fixed Moisture Farming District not being added to oasis, sandsea, and desert worlds.
  • Fixed Mando turret icons.
  • Fixed a few localization issues.
  • Fixed Minor Factions with wrong portraits.
