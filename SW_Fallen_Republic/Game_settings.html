As part of a total conversion mod, the setting that can be set before a game is started has also changed. The game setting enables the player to choose between all the maps created for Star Wars: Fallen republic, as well as influence certain game mechanics. Furthermore, these settings make game sessions shorter or longer, simpler or more complex, and increasingly difficult if desired.

For information about AI difficulty, Scaling difficulty, and Ironman mode please read through the Stellaris Game Settings.

Galaxy

The galaxy settings in Star Wars: Fallen Republic differs from regular Stellaris in that there are no separate sliders for map size and star amount. Instead, all maps are static. Some will feature single planet starts (both fixed and random) and different shaped galaxies.
