The Megastructures from the base game have either been removed or altered. A significant change is that megastructures can be built from the start of the game and do not require an Era Perk. Megastructures require all mining stations to be removed before they can be built. None of the megastructures require DLC.

Contents

  • 1 Gateways
  • 2 Colonies
  • 3 Shipyards
  • 4 Deathstar

Gateways

Infinity Gate construction is unlocked by the Secrets of the Kree special project. They function like normal Gateways.

The Infinity Gate construction site can be built for 100 and 5000 and takes 1080 days.

The Infinity Gate itself can be built for 10000 and 2500 and takes 1800 days.

Deconstruction of the structure can be done at any stage instantly for 50 and −800.

Shipyards

Only civilian vessels and corvettes can be produced at Starbases. Megastructures are required to produce larger ships. All empires start with at least an Orbital Shipyard above their capital planet. Only one megastructure can be built over a planetary body at a time and only one shipyard can be built per system.
