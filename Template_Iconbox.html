Usage

One item in a list of icons (achievements, disasters, etc.). The box name is set as a referenceable anchor on the page, so you don't need to also use id="" to link to it.

{{iconbox|name|description}}
{{iconbox|name|description|image=file}}

Parameters

Parameter Description Required Default
1 Box name and hovertext required none
2 / desc Description optional none
image Image file, ".png" not required optional parameter 1
extension Changes image extension optional png
w Image width, in pixels
Prefixing the number with an x will cause it to set image height instead 		optional default image size
template Uses an assembled template in place of an image optional image
title_font Sets css font/style for title text
set to null for plain text		 optional font-size:larger; font-weight:bold;
