Usage

The colleciton of infoboxes adds the mod header, the navigation for the mod pages, and a version tracker. Unlike the version trackers used for unmodded (SVersion and Version), this does not compare the version to the most recent update. It can also add a disclaimer stating that the page is currently being worked on and is incomplete. 

{{SW Fallen Republic/Navigation|1|2}}

Substitute values into the parameters 1 and 2

Parameters

Parameter Explanation
Patch Which major patch has it been updated for
Is WIP Whether or not the page is currently being worked on

Values

Patch (Parameter 1)

Value Explanation
(patch) The update in the form of a number (ex. 1.0)
timeless This will indicate the article does not change with version
(none) If left blank it will state there is no known version the article is correct for
disable This will disable the display of the version box

Is WIP (Parameter 2)

Value Explanation
WIP A disclaimer is added below the version stating that the article is being worked on
wip Same as above
(none) If left blank it will not show anything about it being actively worked on
