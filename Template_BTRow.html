Template:BTRow

This template builds a row of a {{Bonus table}}.

Usage

{{BTRow| <-- bonus value --> | <-- optional time period: yearly, monthly … -->
| red = <-- if y is set the bonus value will be red -->
| pop= <-- Population -->
| g= <-- Governance -->
| pl= <-- Planet -->
| t= <-- tech -->
}}

pop, g, pl, t must be lists. The parameters are optional. If no value is specified, a hyphen is shown in the table cell.
