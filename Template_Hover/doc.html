Template:Hover/doc

This is the documentation page for Template:Hover.
When it is viewed directly, links using variables may appear broken; do not replace these with hardcoded page names or URLs.

This template creates dotted underlined text with a tooltip.

Parameters

Parameter Description Default Status
1 the text in the tooltip – required
2 dotted underlined text – required
link link=yes enables a link to the dotted underlined text no optional


Example

{{Hover|Technology|Tech}}

Tech 

{{Hover|Technology|Tech|link=yes}}

Tech
