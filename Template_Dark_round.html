This documentation is transcluded from Template:Dark round/doc. (edit | history)

A template to place an icon within a dark circle background.

Usage

Template format:

{{dark round|<file>|<background size>}}

The contents will automatically be set to ×0.75 of their size.

Example

[[File:Mod faction approval.png|24px]] will result in:

{{dark round|[[File:Mod faction approval.png|24px]]|24px}} will result in:
