Star Trek New Civilisations/Terraforming

Game modification: Star Trek New Civilisations

Star Trek New Civilisations expands the Terraforming system to require the creation of appropriate Terraforming Facilities on planets in order activate specific terraforming options.

List of Terraforming Facilities

Name Function
Atmospheric Generation Facility Produces atmosphere
Atmospheric Processing Facility Converts atmosphere
Atmospheric Sequestration Facility Removes atmosphere
Orbital Solar Mirror Provides heat
Orbital Solar Shade Removes heat
Verteron Array Provides water
Hydrolysing Array Removes water
Variable Inertia Array Provides rotation
Weather Control Network Manages weather patterns

Terraforming Grid


Entries with red borders are not able to be colonised, but can be terraformed. Players are able to terraform into an uncolonisable type only if the planet is not colonised.
