Template:Icon link/doc

This is the documentation page for Template:Icon link.
When it is viewed directly, links using variables may appear broken; do not replace these with hardcoded page names or URLs.


This template is used to generate icons with custom links.

Parameters

Parameter Description Required Default
1 Icon used required —
2 Anchor link required —
3 If set to iconify, adds the text of parameter 2 after the icon optional —
iconify
  • If set to yes, appends the text of parameter 2 after the icon
  • If set to full, also adds the text of parameter 1
 optional —
w Sets the size of the icon optional 24px
page Sets / overrides the linked page optional —

Supported icons

Parameter 1 Icon Linked page Notes
anomaly Anomaly# Also supports specific anomaly levels, i.e. for 'anomaly 1', etc.
digsite Archaeological site# Also accepts 'archaeological site' or 'archaeology site'
astral rift Astral rift#
edict Edicts#
policy Policies#
agenda Agenda#
situation Situations#
research speed
  • Physics research#
  • Society research#
  • Engineering research#
 Icon and link used depends on the tech field specified in parameter 2.
  • If parameter 2 does not match any tech field, standard research speed icon will be used.

Examples

Wiki text Produces
{{icon link|policy|Civilian Economy|iconify=yes}} Civilian Economy
{{icon link|edict|Capacity Subsidies|iconify=yes}} Capacity Subsidies
{{icon link|edict|Capacity Subsidies|iconify=full}} Capacity Subsidies edict
{{icon link|research speed|Computing|iconify=full}} Computing research speed
