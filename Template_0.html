This template inserts an invisible (visibility:hidden and color:transparent) zero (0) character. It can be used for padding statistics and other numbers in table columns so that they line up properly. The invisible zero may be replaced by any other string.

Parameter

Parameter Description Default Status
1 invisible string 0 optional
