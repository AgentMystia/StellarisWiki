Template for district icons used on the Districts page

Parameter Description Format
1 District icon, uses an image from Category:District icons. District_[1].png
2 District resource icon, uses an image from Category:District resource icons. District_resource_[2].png
name District name. Also generates a link to the district.
w Icon width/size 77px
