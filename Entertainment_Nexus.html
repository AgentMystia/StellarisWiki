Entertainment Nexus is a semi-unique Astral rift; each empire can encounter it only once per game.

The rift appears after the empire investigates the Bright Object Detected anomaly and chooses to pay Energy.

Summary of Possible Outcomes

These are all of the possible end results for terminating the rift. This table does not include rewards given for intervening chapters.

Chapter Outcome Astral Threads Reward
8-A (Acrobats + Tissue Sample)
  • Receive a random Genetics technology.
  • Receive some Society research.

8-B (Acrobats + Tip)
  • Receive the Wormhole Stabilization technology, if possible.
  • Otherwise, receive 50% progress to a random Particles technology.

  • If the main species is biological and not a Gestalt Consciousness:
    • Receive the Revolutionary Medi-Gel modifier, which grants +25% job efficiency for Farmers. (Appears to apply to Medical Workers instead)
  • If Gestalt Consciousness or non-biological:
    • Receive a random Farming technology.

  • Receive 10% progress to a random Robotics technology.

  • Gain the Spontaneous Crystallization edict, which converts 1% of minerals income into rare crystals.
  • Specimen obtained: Extreme contortionist DNA
8-F (Contortionist + Tip)
  • Receive some Physics research.
