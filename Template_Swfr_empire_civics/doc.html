This is the documentation page for Template:Swfr empire civics.
When it is viewed directly, links using variables may appear broken; do not replace these with hardcoded page names or URLs.

For a generic idea group: 

{{swfr empire civics
|name        = <name of the group>
|events      = <page detailing events associated with the group>
|header      = <omit to use 2nd level heading, or set to h3 to use a 3rd level heading>
|version     = <major patch number for which this is valid>
|civic1       = <name of first idea>
|civic1icon   = <filename (NOT key for Template:Icon) for an icon to use with this idea, omitting "File:" and ".png">
|civic1effect = <effects of first civic>
...
|civic2       = <name of second idea>
|civic2icon   = <filename for an icon to use with this idea>
|civic2effect = <effects of seventh idea>
|bonus       = <bonus for completing the group>
|notes       = <any notes, e.g. prerequisites for taking it>
}}

For a country's idea group: 

{{swfr empire civics
|header      = <omit to use 2nd level heading, or set to h3 to use a 3rd level heading>
|version     = <major patch number for which this is valid>
|name2       = <name of idea group as used in game, e.g. "Hanseatic" for Lübeck, shown on the country page>
|name        = <name of country (including flag) or group, shown on National ideas and Group national ideas>
|tradition1  = <first tradition>
|tradition2  = <second tradition>
|idea1       = <name of first idea>
|idea1icon   = <filename (NOT key for Template:Icon) for an icon to use with the first idea, omitting "File:" and ".png">
|idea1effect = <effects of first idea>
...
|idea7       = <name of seventh idea>
|idea7icon   = <filename for an icon to use with the seventh idea>
|idea7effect = <effects of seventh idea>
|bonus       = <ambitions>
|notes       = <any notes>
}}
