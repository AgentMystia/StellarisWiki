Usage

A wrapper for {{Unsigned}}, it takes input copied and pasted directly from the history tab as the first unnamed parameter.

For example:

  • {{subst:xsign|22:58, 14 August 2010 Example}} is equivalent to {{Unsigned|Example|22:58, 14 August 2010}} and produces:
    — Preceding unsigned comment added by Example (talk) 22:58, 14 August 2010 (UTC)


Preferences must be set to display time in UTC. Otherwise, convert the time to UTC before saving.

Supported formats:

  • 21:48, November 28, 2010
  • 21:48, 28 November 2010
  • 21:48, 2010 November 28

Not supported:

  • 2010-11-28T21:48:15
