Cycle change events
 Available only with the Shadows of the Shroud DLC enabled.

Cycle change events will occur for empires with the Superstitious Beliefs, Fortune Enterprise, Zodiac Nexus and Pattern Finders civics every 13 years when the current cycle ends. Every event can only happen once per game.

Positive events

The chance for the empire to receive a positive event is 50% but if the cycle is Cycle of Fortune the chance will be 75% and if the cycle is Cycle of Omens the chance will be 25%.
