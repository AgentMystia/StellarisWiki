 What if reality is just a game?

Once a player researches the Meta-dimensional Theory technology (under the Physics department), they can add the Black-Hole Signal Transmitter building to starbases orbiting a black-hole and the Pulsar Signal Receiver building to starbases orbiting a pulsar. Their scientists will then try to send a "message" through the black-hole to see if the pulsar is acting as a signal repeater. Affirmative results would enable further research into Meta-physics.

Projects towards Sublimation

Once the Meta-dimension is discovered, scientists start to think about its implications. Would it be possible to "amend" the rules by which the universe works? Is it possible to write a "mod" to the galaxy, and start cheating?

Project name Time Cost Effects Requirements
New Dimension 10
  • Starts random events (once every 90-120 days)
  • Every other empire gets the Meta-dimensional Theory tech option
  • Pulsar Signal Receiver
  • Black-Hole Signal Transmitter
Universal Language 1800
  • Ends the random events chain generated by the player
  • The first empire to complete the project gets the Cheat Code modifier:
  • +0.5% Daily Hull Regeneration
  • +0.5% Daily Armor Regenaration
  • +50% Weapons Range
  • Other empires completing the project get the Minor Cheat Code modifier:
  • +0.1% Daily Hull Regeneration
  • +0.1% Daily Armor Regenaration
  • +10% Weapons Range

New Dimension project completed

Modding Physics research: 100000 Enable the Reboot edict Universal Language project completed


When the mod is ready to go in production, the Reboot edict will be available for the player. Activating it will require to own a fully upgraded Science Nexus. Once activated, the player's empire would totally sublimate into another level of existence (every planet/colony/pop/ship will disappear), the modding backdoor would be permanently sealed (no other sublimation) and the player will be considered the winner of the game.

Another side effect with such an action will be that other Extra-dimensional beings take notice that something happened in the Galaxy, and will take a closer look...

The game shall continue for the remaining players, to see what happens to those left behind...
