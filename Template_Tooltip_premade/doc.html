This is the documentation page for Template:Tooltip premade.
When it is viewed directly, links using variables may appear broken; do not replace these with hardcoded page names or URLs.

Premade tooltips, similar to the ones in the {{Tooltip job}} template.

Contents

  • 1 Parameters
  • 2 Examples
  • 3 Tooltips
    • 3.1 Civics

Parameters

Parameter Description Required Default
1 Premade tooltip template from the list below. required none
2 Sets in-between text between tooltip text and the icon. optional none
text Overrides the underlined text (keeps the icon) optional parameter 1
icon Overrides tooltip icon
  • If set to no, removes the icon.
 optional default
index Allows tooltip-in-a-tooltip functionality, only set to 2 or 3 optional none
exclude Excludes specified entries from the tooltip:
  • If set to gestalt, excludes all gestalt entries
  • If set to hive, excludes all hive entries
  • If set to machine, excludes all machine entries (including individualist machines)
Works only for supported tooltips
 optional none
