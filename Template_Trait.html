Template used for leader traits. See Category:Leader trait icons for the trait images and Category:Leader trait icon elements for the trait background and overlay images.

Parameters

Parameter Description Required Default
1 Image file name, not including "Leader_trait_" or "png" required —
2 Name of the icon, unnecessary if it matches 1 optional [1]
w Size of the icon
Cannot use x24px format		 optional 34px
bg Icon background, uses basic trait background by default:
  • eng / engineering = engineering
  • soc / society = society
  • phy / physics = physics
  • psi / psionic = psionic
  • spec / specialist = specialist
  • neg / negative = negative
  • adv / advanced = advanced (unused in-game)
 optional basic
rarity Trait rarity, used to determine trait frame and overlay.
  • veteran
  • destiny
veteran=yes or destiny=yes can also be used instead.
 optional —
tier Used to indicate trait tier/level
  • 1
  • 2 - automatically adds "II" to trait name in hover text
  • 3 - automatically adds "III" to trait name in hover text
 optional —
council Adds the councilor overlay optional —
class Sets which leader class the trait belongs to, used to determine which page is linked by default
  • official = Official traits
  • scientist = Scientist traits
  • commander = Commander traits
  • default = Common leader traits
 optional Common leader traits
link Sets a custom link optional [class] traits#[3]
hover Sets a custom hover text, overrides tier number. optional [3]
image Sets an alternate image for trait icon optional Leader trait [1].png

Examples

Wikitext Icon
{{trait|Architectural Sense|class=official|rarity=veteran|council=yes}}
{{trait|space miner|Geology Expert|w=34px|veteran=yes|tier=3}}
{{trait|Scholarium Explorer|bg=spec|class=scientist}}
