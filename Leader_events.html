Combat victory random events

These random events can be triggered by the on_fleet_destroyed_perp on_action, which occurs when a fleet defeats another in combat. All seven of these events have a weight of 2, and no event triggering has a weight of 300. This gives each event a chance of 2/314 to 2/302, or approximately 0.64% to 0.66%.

The below description is one of several available for this event.

Captain [officer.GetName], the commanding officer of the [ship.GetName], served with distinction during the recent engagement that took place in the [system.GetName] system. The Admiralty on [Root.Capital.GetName], recognizing the qualities of a promising flag officer, has decided to promote the Captain to the rank of full admiral.
 
Trigger conditions:
  • is_country_type = default

 Is triggered only by:

Galactic Paragons DLC events

This is the list of events found in Stellaris/events/paragon_events.txt.

Every decade there is a roll on a table for potentially triggering one of these events. For non-gestalt empires, the outcomes are:

  • 15% chance: Dysfunctional Council
  • 5% chance: Inspired by Cosmic Phenomenon
  • 10% chance: Confiscated Contraband
  • 10% chance: Unified Council
  • 5% chance: Personal Development
  • 5% chance: Scientific Progress
  • 10% chance: Leaked Information
  • 10% chance: Trendsetter
  • 20% chance: Echo in Eternity
  • 10% chance: Empire-Wide Scandal

For Gestalt Consciousness empires, there is a 10% chance to trigger Idiosyncratic Tendencies for machine/robot leaders.
