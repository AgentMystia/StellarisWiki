Limbo: A Second Chance
Please help with verifying or updating this infobox. It was last verified for version 2.6.

We could give an extinct species of ancient aliens a second chance at life by uploading their neural scans to robotic bodies.

 

Requirements:

  • None
 Cost to Research:
  • 1000 Engineering Research


On Success:
  • Set the country flag "limbo_uploaded_robots".
  • If the following conditions are true:
      • Does not have the policy AI Outlawed
      • There is a habitable, uncolonized planet in the empire's borders with no anomalies
      • There exists a system with the following attributes:
        • Is not owned by any empire
        • Does not have the star flags "hostile_system", "guardian", "enclave", or "primitive_system"
        • There exists a habitable, uncolonized, unowned planet in the system with no anomalies
    • Triggers country event Limbo: a New Beginning (anomaly.189).
  • Else, if the following conditions are true:
      • Does not have the policy AI Outlawed
      • There is a habitable, uncolonized planet in the empire's borders with no anomalies
    • Triggers country event Limbo: a New Beginning (anomaly.1198).
  • Else, if the following conditions are true:
      • Has the technology Droids
      • Does not have the policy AI Outlawed
      • There exists a system with the following attributes:
        • Is not owned by any empire
        • Does not have the star flags "hostile_system", "guardian", "enclave", or "primitive_system"
        • There exists a habitable, uncolonized, unowned planet in the system with no anomalies
    • Triggers country event Limbo: Awakening.
  • Else, if there are no colonizable planets:
    • Triggers country event Limbo: Spirits in the Wires.
