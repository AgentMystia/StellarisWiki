A tooltip template used for alternate versions of a gameplay element, such as job swaps and trait names.

Parameters

Parameter Description
1 Sets the underlined text
  • Optional if either default, individualist, or non-corporate parameter is used

Variants

These parameters will define the variants shown in the tooltip. Unused parameters will be omitted. Civic icons will used the icons defined in {{icon}} template.

Parameter Icon Usage
default Default variant
  • Changes to if any of the gestalt variants are used.
  • Changes to if a corporate variant is used.
individualist Unnecessary in most cases, but will never change icon
non-corporate Unnecessary in most cases, but will never change icon
corporate Corporate variant
gestalt Gestalt variant
hive Hive Mind variant
machine Machine Intelligence variant
extra — Appended at the end of the variant list, does not include icons
