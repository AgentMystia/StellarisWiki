Stellaris Wiki

Template:DLC

The template's main purpose is to provide useful "at-a-glance" info on the related product. Store links are to be provided via the {{Store}} template.

Parameters

Parameter Notes Status
header
  • DLC title (expansions will also display the in-game icon).
  • Expansion icons can be found in Category:Expansion icons.
 required
logo
  • DLC banner/header (usually from Steam for size consistency).
  • DLC banners can be found in Category:DLC banners.
 optional
type DLC type (includes link to section on DLC page). Can be any of:
  • Expansion
  • Narrative expansion
  • Mechanical expansion
  • Story pack
  • Species pack
  • Music pack
  • Cosmetic pack
  • E-book
required
patch
  • Add only for gameplay affecting DLCs.
  • Only major versions - 1.x good, 1.x.y bad
 optional
date Release date in the format YYYY-MM-DD required
store
  • Base product link to the Paradox Interactive / Steam store.
  • To be used alongside the {{Store}} template.
 required
prev / next Adds links to the previous and next DLCs released in order
  • DLC wiki page names must be provided
 Optional

The DLC infoxbox for the Utopia expansion will be set up as follows:
