# Stellaris Wiki (Static Mirror)

A static mirror of [stellaris.paradoxwikis.com](https://stellaris.paradoxwikis.com/),
the community Stellaris wiki, deployed to GitHub Pages for fast, ad-free, archival
reference.

## Contents
- All content pages (main namespace, project, templates, categories): rendered HTML
- All referenced images
- Faithful Vector-Paradox wiki styling (content CSS preserved)

Built from the MediaWiki API (`action=parse`) with edit-links, page history, and
other read-irrelevant MediaWiki chrome stripped out for a clean static site.

## Usage
Browse at the GitHub Pages URL. Red links (pages that don't exist on the source
wiki) link back to the live site.
