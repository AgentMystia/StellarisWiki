Template:File icon/doc

This is the documentation page for Template:File icon.
When it is viewed directly, links using variables may appear broken; do not replace these with hardcoded page names or URLs.

Generates icons from image files and links to an appropriate page with an anchor link. Unlike the {{icon}} template icons, these icons do not need to be manually defined.

Paramaters

The icon uses the [[File:{pre}_{2}.png|{w}|link={page}#{3}|{3}]] format.

Parameter Description Required? Default
1 Defines the default file prefix and linked page. required —
2 Icon name required —
3 Display name and anchor link; optional if it matches parameter 2 optional [2]
w Icon size optional 24px
pre Defines the file prefix optional depends on file type
page Defines the linked page optional depends on file type
type Modifies part of the default link optional depends on file type
iconify Appends the text of parameter 3 optional —

Example

{{file icon|trait|presapient|Serviles|iconify=yes}} results in: Serviles

{{file icon|trait robot|Domestic Protocols|iconify=yes}} results in: Domestic Protocols
