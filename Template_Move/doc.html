Template:Move/doc

This is the documentation page for Template:Move.
When it is viewed directly, links using variables may appear broken; do not replace these with hardcoded page names or URLs.

Move can be used to highlight articles with a request for a name change. The template calls for discussion.

Usage

This table may be added by copy-pasting the template as shown below into an article. 

{{Move| <--insert here the proposed new pagename--> }}

Updating this template

You shouldn't have to.

Example

{{Move|Template:Move}}

Will display this:

It has been proposed below that Template:Move/doc be renamed and moved to Template:Move.
Please participate in the discussion to support or oppose the move on this talk page, usually under the heading “Requested move”.
If, after a few days, a clear consensus for the page move is reached, please move the article and remove this notice, or request further assistance.
